# MediPin ‚Äì Your Nearest care, one pin away

MediPin is a location-based **emergency assistance platform** that helps users quickly find the **nearest hospitals and clinics within a 5km radius**, get **real-time navigation routes**, and follow **step-by-step first-aid instructions** during critical medical situations.

The goal is simple:
When every second matters, **reduce panic and guide users to the fastest medical help**.

---

## üöÄ Features

| Feature | Description |
|--------|-------------|
| **Nearby Medical Facility Locator** | Automatically detects the user's current location and shows nearby hospitals & clinics on a live map. |
| **5 KM Radius Safe-Zone Scanning** | Shows only hospitals/clinics that are realistically reachable within emergencies (5 km surrounding area). |
| **Route Navigation & Turn-by-Turn Directions** | Uses Leaflet Routing to provide live **path guidance** to selected hospitals. |
| **Specialized Category Based Search** | Search by specialty (Dentist, ENT, Cardiologist, Pediatrician, etc.) for more accurate results. |
| **Emergency First-Aid Assistance** | Provides **clear step-by-step guidance** for medical emergencies such as heart attack, shock, burns, heatstroke, and more. |
| **User Reviews & Ratings** | Allows users to provide feedback based on their experience at local medical centers. |
| **Secure Authentication** | Login with traditional credentials or **Google OAuth** for seamless access. |
